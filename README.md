# Candy Smasher & Sharer
### Description
This is a Python tool that helps friends share candies fairly. If the candies can't be shared equally, it calculates how many need to be "smashed" (the leftovers).

### What it does:
It calculates how many candies each friend gets using math.

It finds out how many candies are left over using the modulo operator (%).

 It asks for a friend's name and gives them a nice welcome message.

### How to use it:
Run the script.

It will show you the math for 10 candies and 3 friends.

Type a name to see the welcome message!
